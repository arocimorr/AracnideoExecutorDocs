API Reference

Complete documentation for all available functions in AracProtect executor.

! Security Notice

All file-related functions operate within a sandboxed environment and cannot access files outside of the executor's sandbox directory. This ensures protection against malicious code attempting to access or modify sensitive system files.

HTTP Functions

HttpGet
HttpGet(url) -> string

Performs an HTTP GET request to the specified URL and returns the response body as a string. 

local body = HttpGet("https://example.com")
print(body)
HttpPost
HttpPost(url, body?, userAgent?, timeoutMs?) -> string

Performs an HTTP POST request to the specified URL with optional body, user agent, and timeout.

Default user agent: "aracnidea executor" 

local resp = HttpPost("https://httpbin.org/post", "hello world")
print(resp)

local resp2 = HttpPost("https://httpbin.org/post", '{"a":1}', "MyUA", 8000)
print(resp2)

File System Functions

writefile
writefile(filename, content)

Writes content to a file within the AracFiles sandbox.

Only writes within the AracFiles sandbox directory. 

writefile("demo/hello.txt", "line1\nline2")
readfile
readfile(path) -> string

Reads and returns the content of a file as a string. 

local c = readfile("demo/hello.txt")
print("content:", c)
makefolder
makefolder(path, foldername)

Creates a new folder at the specified path.

Path can be "" for root directory. 

makefolder("", "demo")
makefolder("demo", "sub")
getfiles
getfiles(path) -> string

Returns a list of files and folders at the specified path.

Returns format: {file:: a.txt} {folder:: subdir} ... 

print(getfiles(""))       -- root
print(getfiles("demo"))   -- inside demo
delfile
delfile(path)

Deletes the file at the specified path. 

delfile("demo/hello.txt")
delfolder
delfolder(path)

Deletes the folder at the specified path. 

delfolder("demo/sub")

Script Functions

loadstring
loadstring(code) -> function

Compiles and returns a function from the provided Lua code string. The code is executed via DLL when you call the returned function. 

local f = loadstring([[print("hello from loadstring")]])
f()
